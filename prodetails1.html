Project Title: Face Mask Detection using OpenCV

Project Overview:

This project aims to build a real-time face mask detection system that 
identifies whether a person is wearing a mask or not. It uses computer 
vision and deep learning techniques to analyze live video feed and 
classify detected faces as “Mask” or “No Mask.”

Objectives: 

To detect human faces from a live webcam or image.

To classify each detected face as wearing a mask or not wearing a mask.

To provide visual feedback using bounding boxes and labels in real time.

Tools and Technologies Used:-

Programming Language: Python

Libraries & Frameworks:
    
OpenCV – for image processing and face detection

TensorFlow / Keras – for building and training the CNN model

NumPy – for array and numerical operations

Matplotlib – for visualization (optional)
IDE:
Jupyter Notebook / VS Code
Dataset:
Publicly available Face Mask Detection Dataset (Kaggle)

Working Process / Implementation Steps:

    1. Data Collection:
      2. Gathered images of people with and without masks from an open-source dataset.
    2. Data Preprocessing:
      • Resized and normalized images.
      • Labeled images as “Mask” or “No Mask.”
    3. Model Building:
      • Built a Convolutional Neural Network (CNN) using TensorFlow/Keras.
      • Trained the model to distinguish between masked and unmasked faces.
    4. Integration with OpenCV:
      • Used OpenCV’s Haar Cascade or DNN for real-time face detection.
      • Passed the detected faces to the trained model for mask prediction.
    5. Output Visualization:
      • Displayed results with green boxes for “Mask” and red boxes for “No Mask.”
      • Showed live video feed with accurate label updates.

    • Results:

    • Achieved high accuracy in real-time mask detection.
    • Successfully tested on live webcam video and static images.
    • Demonstrated effective use of AI and computer vision for public safety applications.
